Project Summary

Feb 18, 2020Hits

这是一个百科聊天机器人，用户可以在聊天框中询问聊天机器人任何问题，聊天机器人会将问题传回云端，云端服务其则调用成熟的自然语言处理算法进行处理后返回回答结果，本地程序再一次将结果数据进行处理后则在回复框中进行显示。
